Proses dan Tahap Pembuatan Website Profesional

Proses dan Tahap Pembuatan Website Profesional

Posted by Admin
Deskripsi

Panduan bagi PR / Corcomm perusahaan. Artikel saya kali ini membahas mengenai perencanaan pembuatan atau redesign website perusahaan. Jika anda adalah tim public relation atau corporate communication, atau orang yang bertanggungjawab dalam perusahaan yang akan membangun web untuk perusahaan anda, maka artikel ini akan membantu Anda.

1. Rumuskan Tujuan Website

Pada tahap awal ini, Anda perlu mengidentifikasi tujuan akhir website yang akan dibangun. Biasanya diperlukan kerjasama erat dengan pemangku kepentingan lainnya, seperti bagian marketing, produk, produksi, dan tentunya jajaran manajemen.

Pertanyaan-pertanyaan ini akan membantu Anda merumuskan tujuan pembuatan website:

A. Untuk siapa website ini, siapa yang akan menjadi audiencenya?

B. Apa yang mereka harapkan untuk temukan atau lakukan pada website ini?

C. Apakah tujuan utama website ini adalah informasi profil perusahaan, atau katalog promosi produk?

D. Identifikasi keunggulan perusahaan, layanan atau produk. List-down differensiasi/ unique selling point perusahaan di tengah industri.

E. Referensi web, dan apa website pesaing, dan apa yang ingin Anda ambil atau bedakan dari pesaing tersebut?

F. Jika Anda me-redesign website lama Anda, apa yang ingin Anda ubah atau perbarui?

Tahap perumusan tujuan adalah bagian terpenting dari proses web design. Jika pertanyaan-pertanyaan ini tidak dijawab dengan jelas dalam ringkasan, seluruh proyek pekerjaan dapat berangkat ke arah yang salah, dan menghasilkan website yang tidak sesuai dengan kebutuhan.

Akan sangat berguna untuk membuat list poin jawaban yang teridentifikasi dengan jelas, atau ringkasan satu paragraf tentang tujuan yang diharapkan. Ini akan membantu menempatkan pekerjaan website pada arah yang benar, dan membantu tim web design agency bekerja secara optimal.

2. Susun Struktur dan Lingkup Website Brief

Setelah merumuskan tujuan website, Anda harus menyusun struktur web dan brief lingkup website. Struktur website merupakan dasar pijakan bagi proses web design yang baik.

Struktur website berbentuk sitemap website; kerangka susunan menu dan informasi. Sitemap akan memberikan gambaran jelas bagi web agency Anda memahami arsitektur informasi dan hubungan antar menu dan elemen.

Tanpa membuat sitemap, Anda seperti membangun rumah atau bangunan tanpa blueprint perencanaan ruang. Seperti blueprint, sitemap web harus bersifat final, tidak berubah-ubah.

Selain membuat sitemap, Anda juga harus mendeskripsikan lingkup website. Ruang lingkup sederhana di antaranya penyusunan fitur yang ingin dibuat, perencanaan bentuk konten yang ingin ditampilkan mengacu pada sitemap.

Misalnya :
"Pada menu product, terdapat fitur katalog produk dan fungsi search."
"Pada menu Investor > Financial Highlight, terdapat konten berupa download file pdf summary laporan keuangan."

List-down sitemap dan lingkup website Anda dalam satu file yang lengkap dan jelas.

Di tahap ini, pekerjaan akan menyita perhatian Anda. Tapi percayalah, data yang Anda susun akan sangat dibutuhkan nantinya.

3. Siapkan Brand Identity Brief

Website perusahaan Anda harus mewakili identitas perusahaan. Cara terbaik untuk menampilkan identitas perusahaan adalah dengan membuat tampilan website yang in-line atau senada dengan identitas brand perusahaan.

Web designer yang bagus akan membutuhkan dan meminta panduan mengenai brand identity guideline perusahaan Anda. Anda harus mempersiapkan file materi ini.

Brand identity guideline biasanya setidaknya akan meliputi :
File Logo. Sebaiknya Anda mempersiapkannya berupa raw file. Yaitu file mentah logo yang biasanya berformat Adobe Illustrator, ataupun Photoshop. Designer akan mengektrak elemen dari file tersebut ke dalam elemen design untuk menampilkan identitas perusahaan.

Panduan brand. Panduan brand biasanya berupa file logo guideline. Berisi aturan penggunaan logo, kode warna yang digunakan, dan elemen-elemen logo. Kode warna akan membantu web design agency Anda untuk membangun tone web design yang sesuai.

4. Kumpulkan dan organisasi Konten

Konten merupakan bagian yang utama dari website Anda. Website yang Anda bangun adalah media untuk menampilkan konten Anda. Jadi, idealnya konten ditentukan terlebih dahulu sebelum membangun design web untuk mewadahinya.

Konten harus dipersiapkan sesuai dengan struktur / sitemap website. Jika tidak, maka web designer Anda belum tentu akan mampu memahami apa yang diperlukan pada satu halaman website.

Misalnya, akan berisi apa halaman laporan keuangan website Anda? Tanpa ide konten yang jelas, web designer akan bertanya-tanya apakah konten berupa gambar, video, file download, teks artikel, atau kombinasi. Ketidakjelasan ini akan mungkin membuat proyek pembuatan website Anda berlarut-larut dengan revisi.

Ada kalanya konten memang belum tersedia ketika pekerjaan web design sudah harus dimulai. Hal ini karena mengumpulkan konten bukanlah perkara mudah. Konten sering harus berasal dari berbagai divisi berbeda dari perusahaan. Butuh waktu dan kordinasi yang baik dari semua bagian untuk menyuplai konten yang berkualitas.

Jika ini terjadi, maka setidaknya Anda harus merumuskan tiap bentuk konten per halaman, dan mempekerjakan hanya web design agency atau jasa pembuatan website profesional yang berkualitas, dengan konsultan yang smart untuk menerjemahkan kebutuhan Anda.
